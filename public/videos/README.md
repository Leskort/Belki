# Видео для фона

Поместите видео файл с названием `forest-background.mp4` в эту папку.

Рекомендации:
- Формат: MP4
- Разрешение: 1920x1080 или выше
- Длительность: 10-30 секунд (будет зациклено)
- Размер файла: старайтесь держать до 5-10 МБ для быстрой загрузки

Примеры тем для видео:
- Лес с ёлками
- Зимний лес
- Снегопад в лесу
- Тёмный лес с неоновым освещением

Если видео не загружено, будет показан градиентный фон как fallback.
